## Security-first reading path

If you are new to Diogel, read these first:

1. [Key Management](./key-management.md)
2. [Settings](./settings.md)
3. [Dashboard](./dashboard.md)
4. [Wallet Connections](./wallet-connections.md) — if you plan to use Nostr Wallet Connect or Lightning payments.

## Troubleshooting path

- Website login/signing issue: [Event History](./event-history.md), then [Key Management](./key-management.md).
- Profile issue: [Profile Management](./profile-management.md), then [Relay Management](./relay-management.md).
- Publishing issue: [Dashboard](./dashboard.md), then [Relay Management](./relay-management.md).
- Wallet issue: [Wallet Connections](./wallet-connections.md).
- Backup/import issue: [Settings](./settings.md).
